arxiv.submission.config module¶

Submission core configuration parameters.

arxiv.submission.config.ENABLE_ASYNC = '0'¶

If 1, asynchronous callbacks will be dispatched to the worker.

Otherwise they will be executed in the thread in which they are called.

arxiv.submission.config.ENABLE_CALLBACKS = '0'¶

If 0, callbacks bound to events will not be executed.

arxiv.submission.config.URLS = [('submission', '/<int:submission_id>', 'submit.arxiv.org'), ('confirmation', '/<int:submission_id>/confirmation', 'submit.arxiv.org')]¶

URLs for external services, for use with flask.url_for(). This subset of URLs is common only within submit, for now - maybe move to base if these pages seem relevant to other services.

For details, see arxiv.base.urls.
