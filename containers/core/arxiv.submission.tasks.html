arxiv.submission.tasks module¶

Provides support for asynchronous tasks.

arxiv.submission.tasks.get_or_create_worker_app()[source]¶

Get the current worker app, or create one.

Uses the Flask application global to keep track of the worker app.

Return type:Celery
arxiv.submission.tasks.is_async(func)[source]¶

Turn a function into an asynchronous task.

Registers the function with the worker application, and decorates the function with logic to dispatch the function to the worker when called. When the decorated function is called, a task is added to the worker queue and an empty iterable is returned. If ENABLE_ASYNC=0 on the app config, calls to the decorated function will execute in-thread and return normally.

Return type:Callable
